# Codebase Map

## Important Folders

- `app/(dashboard)` - dashboard routes.
- `app/api` - API routes.
- `app/components` - page and feature components.
- `app/ui` - dashboard shell and CSS.
- `app/lib` - app-level helpers.
- `lib` - shared backend helpers.
- `prisma` - Prisma schema, migrations, seed.
- `obsidian` - this vault.
